Screen Navigation Flow
Discover
Map
Shortlist
Calendar
Settings
FROM DISCOVER TAB
Tap school card → school/[id] (K-12 detail)
Tap HEI card → institution/[id] (HEI detail)
Tap "Compare Selected" bar → compare/
Tap filter button → FilterSheet (bottom sheet overlay)
"View on Map" from detail → map tab (with pin focused)
FROM MAP TAB
Tap pin → bottom sheet preview (low snap)
"View Profile" from preview → school/[id] or institution/[id]
Tap filter button → FilterSheet (same component)
FROM SHORTLIST TAB
Tap school card → school/[id] or institution/[id]
Tap "Compare Selected" bar → compare/
Tap status stage → updates MMKV (inline, no navigation)
FROM CALENDAR TAB
Tap event → event/[id] (view/edit)
Tap "+" FAB → event/create
Tap "Add to Calendar" → expo-calendar (system intent)
Tap bell → expo-notifications schedule
Overlay Components (not routes -- rendered in-place)
FilterSheet
Single component, renders level-specific sections. Used by Discover + Map tabs. Bottom sheet (50% / 92% snap).
MapPreviewSheet
Compact bottom sheet on pin tap. Low snap (~220pt). Shows school name, address, badges + "View Profile" CTA.
CompareBar
Sticky bar above bottom nav. Shows when 2-3 schools selected. Animates in/out with Reanimated.
OfflineBanner
Non-blocking top banner when NetInfo detects no connectivity. Dismissible.

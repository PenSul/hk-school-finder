Changes from v1

  • schools: Added school_level_tc, district_tc, finance_type_tc, religion_tc, session_tc, students_gender_tc (6 new columns)
  • ugc_programmes: Added level_of_study_tc, mode_of_study_tc (2 new columns)
  • TC columns store the Chinese text from the API's *_TC fields -- used when language is set to Traditional Chinese
  • Filtering always queries against the EN columns; TC columns are display-only (avoids duplicate indexes)
